GoldFind -- Menu - File "Open Archive"

Here you can use four Methods to handle News

* Hamster

In the opened Explorer search for your Hamster-Directory, where is your DB and select the .ini 'Hamster.ini'. With this you are using the DB directly from your Hamster.

*Hamster-OLE

For using OLE you have to set the name of the OLE-Server. Normally this is "Hamster.App", but with some hex-change (see old ReadMe.pdf!), this may be set to anything. You will need a Hamster Classic of at least 1.3.23.0, not started as a service.
