GoldFind -- Menu - File "Search"

    
Here, in the following Input-Box, you can fill in, what you want to find in any form you will have.

Time    - You fill in the time-area, GF searched.


Options - You can select in Order:

   Search Body   - To search in Bodys too/or only
   Search Header - To search in Header too/or only
   Case sensitiv - Writeform will exactly considerated

Search  - Fill in, whatever you search

In 'Search'-Field, there are many ways, to fill in a term - For one example, you can fill in a term as you want to write. If you search for My Birthday, it will find the word My also the word Birthday.

A second method is, to write your term in "" (e.g. "My Birthday").
This will exactly find the word-order you fill in.

A third method is to use RegExp - For using 'RegExp', search for a Site in Internet - One Site, as example in german is http://regexp-evaluator.de/
in english http://www.regular-expressions.info/


To use RegExp it is recommended, you read some special Sites.

All search-patterns you add in this way are searched as "OR"-criterions. Afterwards all found articles are again be searched for patterns beginning with a "+" (plus), these are "AND"-criterions. As the last step, the now concerned articles are searched for the patterns beginning with "-". Only those articles NOT containing any of these "NOT"-criterions, are found.


A click on <Begin> will start your search.
