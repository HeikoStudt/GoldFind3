GoldFind -- System Requirements

- You need any NNTP-Server or Unix-MBX-Files. Preferrable you have the local mail+newsserver Hamster installed.

- You need to have Windows 95 up to XP. Vista-Support is not tested. Linux-Support is not planed.

- You do NOT need to install anything, GoldFind do not change data out of his directory (besides the things *Windows* saves in registry).
